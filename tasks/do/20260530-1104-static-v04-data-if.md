# static v0.4 data IF

状態: do

## 背景

`.workspace/plan-20260530.txt` は v0.4 準拠の P0 として STATIC-001〜008 の正式サポートを求めている。現 main は `/data/latest-manifest.json` と `/data/v/{export_version}/public/...` の immutable export を中心にしており、v0.4 の alias path である `/data/home.json`、`/data/videos/index.json`、`/data/videos/{video_id}.json`、`/data/tags.json`、`/data/calendar/{year}.json`、`/data/artifacts/...` とは差分がある。

## 目的

既存の versioned export を維持したまま、v0.4 の STATIC-001〜008 alias path を materialized JSON として正式に出力し、public contract checker で検証できるようにする。

## タスク種別

機能追加

## スコープ

- static exporter が `/data/home.json`、`/data/videos/index.json`、`/data/videos/{video_id}.json`、`/data/tags.json`、`/data/calendar/{year}.json`、`/data/artifacts/wordcloud/{video_id}.json`、`/data/artifacts/timestamps/{video_id}.json` を出力する。
- `/data/v/{export_version}/public/...` の immutable path は維持する。
- `/data/latest-manifest.json` に v0.4 alias path と対応する versioned path/checksum を含める。
- `tools/check-public-contract.mjs` と tests で STATIC-001〜008 を検証する。
- README と traceability を更新する。

## 非スコープ

- PNG wordcloud 生成はこの task では行わない。v0.4 の `{png|json}` のうち JSON artifact を先に正式サポートし、既存 SVG は互換 artifact として維持する。
- CloudFront 実環境 smoke と dev deploy rehearsal はこの task では行わない。
- API-007 や UI の calendar 画面追加は別 task とする。
