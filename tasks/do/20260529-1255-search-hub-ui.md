## 受け入れ条件

- ホームに大きな検索バーがある。
- quick chips が表示され、検索やタグ選択へ反映される。
- 最近検索が表示され、選択すると検索へ反映される。
- 最新アーカイブが初期画面で確認できる。
- タグ導線があり、タグ選択状態が分かる。
- モバイル幅で片手操作しやすい tap target と配置になっている。
- 本番 UI に架空値・demo fallback を混ぜない。
- 変更範囲に見合う検証と `npm test` が成功する。
