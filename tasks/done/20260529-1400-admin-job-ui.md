# 管理job UI強化

状態: done

## 背景

`.workspace/plan-20260529.txt` の P3-05 に従い、管理 UI から token/CSRF 入力、job 起動、job 一覧、job 詳細、JobEvent、quota usage、static export 起動結果を見られるようにする。

## 目的

既存の管理 API を使い、運用者がブラウザ上で job 起動後の状態・event・quota を確認できる管理 panel にする。

## 計画

1. 既存 admin form と API response shape を確認する。
2. token/CSRF、job 起動、job 一覧、quota usage の既存 UI を崩さず、job 詳細入力と表示を追加する。
3. static export 起動結果として job_id、状態、dry_run、deduplicated を表示し、詳細取得へつなげる。
4. JobEvent を list 表示する。
5. 検証、作業レポート、commit、PR、受け入れ条件コメント、セルフレビューを完了する。
